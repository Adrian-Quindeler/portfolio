Loja Online

Esta é uma loja online desenvolvida para venda de jogos, consoles e peças de computador, com uma interface intuitiva que permite aos usuários navegar facilmente por diferentes categorias de produtos. O sistema foi construído com HTML, CSS, JavaScript e Bootstrap para garantir um layout responsivo, e utiliza Django com SQLite como banco de dados, facilitando o gerenciamento e atualização dos produtos.

⚠️ Para melhor visualização das imagens, gire o celular na horizontal.

Tela Inicial

Apresenta a home da loja com banners promocionais em destaque e categorias principais para fácil navegação.

Catálogo de Jogos

Lista completa dos jogos disponíveis, com visual limpo e organizado para facilitar a escolha do cliente.

Jogos Populares

Exibe os títulos mais vendidos ou bem avaliados, ajudando o cliente a descobrir os mais procurados da loja.

Catálogo de Consoles

Mostra todos os consoles disponíveis para compra, sem filtros aplicados, incluindo marcas e modelos variados.

Lançamentos de Consoles

Apresenta os consoles mais recentes adicionados à loja, ideal para clientes que buscam novidades do mercado.

Administração e Banco de Dados

Visualização da área administrativa onde os produtos são cadastrados, editados e organizados via Django e SQLite.

Logo da Loja

Identidade visual da G4G (Galaxze for Geeks), reforçando a temática geek da marca.
