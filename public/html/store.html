Loja Online

Projeto de uma loja online de jogos, consoles e peças de computador que oferece uma interface amigável para os usuários navegarem por diferentes categorias de produtos. Feito com bootstrap para o layout responsivo, além de html, css e javascript. O sistema é usa o banco de dados padrão do Django, SQLite, para gerenciar o catálogo de produtos, permitindo fácil atualização e manutenção.
